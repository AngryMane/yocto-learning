    その他(OS以外)のパッケージをビルドする

    その他(OS以外)のパッケージをビルドしてみましょう。まだpokyリポジトリをcloneしていない場合、cloneします
    使用するブランチはこちらから選んでください。ここではkirkstoneブランチを選択しています 

    $ git clone https://git.yoctoproject.org/git/poky -b kirkstone
$ cd poky

    次に、その他(OS以外)のパッケージをビルドします。ここでは、busyboxというソフトウェアをビルドします
    yocto/bitbake/pokyとはの通り、以下の手順でビルドします

    1. ビルド環境を設定するスクリプト を実行する
    2. bitbakeコマンド を実行する
    # ビルド環境を設定するスクリプト を実行する
$ source oe-init-build-env

# bitbakeコマンド を実行する
$ bitbake busybox
