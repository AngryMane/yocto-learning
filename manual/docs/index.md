# はじめに

## 対象読者

以下の全ての条件を満たす読者を想定しています  

* yoctoでやりたいことがある
* Linuxでのソフトウェア開発に慣れており、シェルでコマンドを実行することに抵抗がない

## 本ドキュメントの読み方

[目次](#目次) を上から順番に読んでください  
ただし、各セクション内の `ユースケースの実践方法` のページは興味がなければ飛ばしても良いかもしれません
